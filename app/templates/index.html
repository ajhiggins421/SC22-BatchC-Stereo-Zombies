Technology

Some info about the technology this project uses

GPT-2

GPT-2 is an AI created by OpenAI that translates text, answers questions, etc. in a way that is very similar to humans

PYTHON

Programming language with many modules such as AITEXTGEN and FLASK. This was used to train our models, handle data and host the site

AITEXTGEN

Python library that allows finetuning of GPT-2 models and text generation using said models

FLASK

A micro-framework written in Python that is used to create web applications quickly and easily

NLP

Natural language processing (NLP) is the ability of a computer program to understand human language as it is spoken and written

HTML, CSS, BOOTSTRAP

Front end tools used to build the user interface which allows users to generate text

Timeline

Timeline mapping the process of our project's development

  • Week 1

    During week one we learned the basics of machine as well as deep learning and useful data analysis tools like word2vec and WordCloud. We then planned out and organized our ideas for the project. First we practiced and learned how to use NLP and aitextgen in order to collect data, train our models, and generate the text based on the models. We also learned the basics of python, which allowed us to utilize the amazing modules it provides. With this information, we made a mini project to get a better grasp of these concepts. Then we began to brainstorm what projects we could do and decided on a story generator. After generating our ideas we wrote out what the specs of this product would look like.

  • Week 2

    During week two, we mainly focused on training and gathering data for our models. To start the week, we had a few lessons on how to feed data to our models and how to use RNN, LSTM, Seq2Seq models and Transformers. We then went off on our own to find and scrape data sets of various potential genres we could use. After we found data sets to train our models on, we spent 30-50 minutes on each model training, with many tweaks to the data input and training steps with hopes of improving the fluency of the generated text as much as possible.

  • Week 3

    After we got our models trained, we began to learn how to properly use html, css, and bootstrap (a css framework). We use this website to document our journey and the steps we took to make our project. We have many sections that describe our technologies used, timeline, information about our team, how the model works and multiple ways to get in touch with AI CAMP via social media handles. We made a model page that incorporates our backend and frontend to allow a user to generate a story, given a prompt and genre.
